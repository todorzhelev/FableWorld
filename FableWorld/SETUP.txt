How to setup the project:
1. Download and install Visual studio 2015 Community Edition or Enterprise

2. Download and install Microsoft DirectX SDK (June 2010) https://www.microsoft.com/en-us/download/details.aspx?id=6812

3. Download and install fmod (for the music in the game) http://www.fmod.org/download/ FMOD Studio Programmerâ€™s API and Low Level Programmer API

4. Open FableWorld.sln

5. In the project properties in VC++ Directories:
	- in Include Directories add the path to include directories for fmod, directx, src folder of lua and boost
		C:\Program Files (x86)\Microsoft DirectX SDK (June 2010)\Include
		C:\Program Files (x86)\FMOD SoundSystem\FMOD Programmers API Windows\api\inc
		F:\PROJECTS\FableWorld\FableWorld\CORE\CORE\libs\boost_1.59.0
		F:\PROJECTS\FableWorld\FableWorld\CORE\CORE\libs\lua-5.1.4\lua-5.1.4\src
		
	- in Library Directories add the path to library directories for fmod and directx
		C:\Program Files (x86)\Microsoft DirectX SDK (June 2010)\Lib\x86
		C:\Program Files (x86)\FMOD SoundSystem\FMOD Programmers API Windows\api\lib
	
6. Press Apply and OK

7. Set the configuration to Debug and start the project
